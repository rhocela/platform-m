# Log in

Access the Pseudonymisation Portal to start managing pseudonymisation projects and generating tokens.

1. Open the Pseudonymisation Portal on your web browser: https://pseudo.lnds.lu/. 
2. Select **Login** and select your authentication method.
3. Enter your credentials.
4. Complete authentication.

You'll be directed to the portal dashboard where you can access organisations, projects, and requests.

:::tip
If you don't have credentials, contact your system administrator to create an account.
:::
